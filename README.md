# Software Services

To run the project do the following

1. Clone the project from GIT using Git CLI
2. cd project
3. cd Software-Service
4. cd backend
5. ./pocketbase serve
6. open new terminal and go to cd project/Software-Service
7. npm i
8. npm run start
